```
git clone iamandydev/<repo-name>
```
(usando HTTPS)
1. Clonar el repositorio en el que se va a trabajar.

```
cd <repo-name>
```
2. Entrar en la carpeta del proyecto.

```
git add .
```
3. Para Seleccionar los archivos modificados.

```
git commit -m "message"
```
4.  Para añadir un comentario que permita identificar los cambios realizados.

```
git push origin <branch-name>
```
5.  Confirma y envía los cambios realizados y los envía desde el repo local y la rama actual, al repo remoto y la rama especificada.

```
git pull
```
6.  Para traer los cambios actualizados del repo remoto y fusionarlos.
